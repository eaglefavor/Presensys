🧪 [testing improvement] Add tests for exportToCSV utility

🎯 What: The testing gap in src/lib/ExportUtils.ts for the exportToCSV utility has been addressed by writing a new set of tests covering this functionality. Since this relies on a DOM implementation for downloading the created CSV blob, I mocked the `Blob` and basic DOM implementations (`document.createElement`, `URL.createObjectURL`, etc) within the `node:test` framework.

📊 Coverage:
- Simple CSV export without metadata
- Full metadata insertion (faculty, department, level)
- Partial metadata insertion
- Empty data array handling

✨ Result: Replaces manual testing of CSV export files with a verified, deterministic test suite that captures regressions and properly verifies correct PapaParse functionality along with the unparsed metadata generation logic before output generation.
